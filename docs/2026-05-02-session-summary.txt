         a MatchPill on each /for-you poster.

PR #136  Recommender v3 (sub-project #37) — patches the math expert's
         review of algo.md. Drops μ at scoring time (was being applied
         twice, compounding to μ² → 36× spread). Adds length penalty
         (sqrt(tag count)). Smooths + clamps IDF for small-catalog
         stability. Adds aversion vector so explicit dislikes
         actively suppress. Replaces calibrated % pill with rank-
         percentile bands.
